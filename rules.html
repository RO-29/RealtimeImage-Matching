Game Rules

  1. You need another opponent paired to play this game (Max 2 at a time).
  2. You may not know whom you are paired with.
  3. During the game play each paired player is shown the same question.
  4. Each question would have primary image and a set of secondary images to match to the primary image
  5. Both paired players can move to the next question only when they choose the same set of secondary images
